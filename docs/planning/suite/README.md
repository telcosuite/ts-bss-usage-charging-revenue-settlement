## Cross-App Flow

Customer and account data combine with catalog, pricing, campaign, consent, serviceability, and recommendation context during CPQ. Accepted quotes or carts create product orders. Product orders decompose into OSS fulfillment work. Billing, charging, usage, payments, credit, fraud, collections, and revenue settlement then manage the financial lifecycle.

The original concise app overview files remain beside these folders and are linked from each app detail pack.
